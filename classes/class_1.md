## class Table
---
### attributes:

  * height (integer)
  * color  (string)
  * people_sitting (array)
  * is_dirty (boolean)  
  * table_number (integer)
  * location (string)
---
### methods:  
  * clean (changes is_dirty to false)
  * seat_person (adds person to people_sitting) 
  * paint (changes color)
  * move (changes location)
